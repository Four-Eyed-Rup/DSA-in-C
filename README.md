# Data Structures and Algorithms in C

This repository contains implementations of fundamental Data Structures 
and Algorithms in C language.

- Linked List
- Stack
- Queue
- Recursion
- Sorting Algorithms
- Searching Algorithms
- Trees
- Graphs
- Dynamic Programming

GCC

## How to Run

gcc filename.c -o output
./output
